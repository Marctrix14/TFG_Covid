El dataset "torax" conté imatges dels pulmons predits després d'aplicar la v_2ROIs de detecció de pulmons
respecte el dataset original d'imatges de radiografia de tòrax.

El dataset "pulmonsRetallats" conté imatges dels pulmons predits després d'aplicar la v_2ROIs de detecció de pulmons
respecte el dataset original processat amb imatges retallades dels pulmons resultants de l'aplicació de la versió d'1 ROI d'en Robert.
